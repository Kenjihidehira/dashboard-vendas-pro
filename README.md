# Dashboard Vendas Pro

Dashboard comercial avançado feito com HTML, CSS e JavaScript puro.

## Funcionalidades

- KPIs comerciais calculados em tempo real
- Filtros por período, canal, equipe e busca
- Gráfico de vendas por dia sem bibliotecas externas
- Acompanhamento de metas
- Tabela de pedidos recentes
- Exportação CSV
- Tema claro/escuro salvo no navegador
- Layout responsivo

## Como rodar

Abra o arquivo `index.html` no navegador.

Se quiser servir localmente:

```bash
npx serve .
```

## Tecnologias

- HTML5
- CSS3
- JavaScript
- localStorage
- Blob API para exportação CSV
